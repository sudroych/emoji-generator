# 🎲 Random Emoji Generator

A simple, fun, and beautiful web application that displays a random emoji every time you click a button!

## ✨ Features

- 🎨 **Beautiful gradient design** with smooth animations
- 🎲 **1000+ emojis** to choose from
- 📊 **Click counter** to track how many emojis you've generated
- 💫 **Smooth animations** when displaying new emojis
- 📱 **Responsive design** - works on all devices
- 🚀 **No dependencies** - pure HTML, CSS, and JavaScript

## 🚀 How to Use

1. **Open the file**: Simply double-click on `index.html` to open it in your web browser
2. **Click the button**: Press the "Generate Emoji" button
3. **Enjoy**: Watch as a random emoji appears with a smooth animation!
4. **Keep clicking**: Generate as many emojis as you want - the counter keeps track!

## 🎨 Customization

Want to customize the app? Here are some easy modifications:

### Change Colors
Edit the gradient in the CSS (line 17):
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

### Add More Emojis
Add your favorite emojis to the `emojis` array in the JavaScript section (line 113)

### Change Button Text
Modify the button text in the HTML (line 95):
```html
<button class="emoji-button" id="generateBtn" onclick="generateEmoji()">
    Your Custom Text Here
</button>
```
